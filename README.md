# Intelligent Feedback & Sentiment Analysis System

## Overview

The Intelligent Feedback & Sentiment Analysis System is an AI-powered web application that analyzes user feedback and classifies sentiments as Positive, Negative, or Neutral. The project uses Natural Language Processing (NLP) and Machine Learning techniques to help organizations understand customer opinions and improve decision-making.

## Features

* Real-time sentiment analysis
* Feedback classification using AI/ML models
* Interactive dashboard with charts
* Responsive and modern UI
* REST API integration
* Issue and performance analysis
* User-friendly interface

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/intelligent-feedback-system.git
cd intelligent-feedback-system
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

## Usage

1. Open the frontend application in the browser.
2. Enter feedback text in the input field.
3. Click the Analyze button.
4. View sentiment results and dashboard analytics.
